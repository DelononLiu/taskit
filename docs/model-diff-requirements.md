# ModelDiff 需求文档

> 神经网络模型精度差异分析工具

---

## 1. 产品定位

一个**桌面工具**（不是网页 App），用于分析同一个模型在不同推理框架下的精度差异。

对标：TensorBoard、Grafana、VS Code —— 打开即用，没有"首页→结果页"的步骤感。

---

## 2. 核心功能流程

```
用户操作：             界面响应：
──────────────────────────────────────
1. 打开工具            → 空状态，显示拖拽上传区（小巧的）
2. 拖拽/选择 .onnx     → 工具栏显示模型信息（文件名、大小、已上传）
3. 勾选比对框架        → 工具栏显示已选框架（TRT/OV）
4. 点击运行            → 主区域显示进度，运行完成后显示结果
5. 查看精度差异        → 主区域显示：指标卡 + 图表 + 层表格
6. 点击某层            → 右侧面板显示该层详细精度指标
7. 切换框架            → 表格和图表切换为该框架的数据
```

**关键原则：所有操作在同一个界面完成，没有页面跳转。**

---

## 3. 布局结构

```
┌──────────────────────────────────────────────────────────────┐
│  [Logo] ModelDiff  [选模型]  model.onnx  [TRT][OV]  [▶运行]  │  ← 工具栏（永远可见）
├─────────────────────────────────┬────────────────────────────┤
│                                  │                            │
│  主区域（flex-1）                 │  右侧面板（380px，有数据时显示）│
│                                  │                            │
│  空状态：                         │  （未选层时干净）            │
│    小型拖拽上传条                  │                            │
│                                  │  （点击表格行后）            │
│  运行中：                         │  层详情：                   │
│    进度指示                       │    - 层名/类型/形状         │
│                                  │    - 精度指标（4项）         │
│  有结果时：                       │    - 框架对比表格           │
│    - 4个指标卡（横排）             │    - 输出值对比图           │
│    - 2个图表（并排）               │                            │
│    - 层表格（等宽字体+色条）        │                            │
│                                  │                            │
└──────────────────────────────────┴────────────────────────────┘
```

### 布局要点
- 全屏铺满，没有 max-width 限制
- 主区域和右侧面板永远在一个视口内，不弹窗不遮罩
- 右侧面板只在选中层时出现，不选中时干净隐藏
- 工具栏高度 48px（h-12），紧凑

---

## 4. 视觉风格

### 4.1 整体方向
专业 ML 工具风，参考 TensorBoard 暗色主题

### 4.2 颜色
- 底色：深色（`#0f0f11` 级别）
- 卡片表面：略浅的深色（`#18181b` 级别）
- 文字：浅灰（`#a1a1aa`）到白（`#fafafa`）
- 通过色：绿色（`#22c55e`）
- 失败色：红色（`#ef4444`）
- 基准色（ONNX Runtime）：蓝（`#1677ff`）
- TensorRT：紫（`#9333ea`）
- OpenVINO：橙（`#f97316`）
- 不支持亮色主题作为默认

### 4.3 字体
- 所有**数字和指标**用等宽字体（`font-mono`）
- 标签用小号大写（`text-[11px] uppercase tracking-wider`）
- 非数据文字用系统无衬线体

### 4.4 间距（重要）
- **紧凑**，非"呼吸感"
- 卡片内边距 `p-3` / `p-4`
- 表格行高约 36px
- 工具栏高度 48px

### 4.5 圆角
- 小元素 4px（`rounded`）
- 面板 0（不要圆角）
- 按钮 4-6px

## 5. 组件清单

### 工具栏（Toolbar）
| 元素 | 说明 |
|---|---|
| Logo | ModelDiff + 图标 |
| 选择模型按钮 | 触发文件选择器 |
| 模型信息 | 文件名（等宽）+ 大小 + "已上传"标签 |
| 框架选择 | TRT/OV 两个按钮，可多选，选中高亮 |
| 运行按钮 | 点击触发推理 |
| 当前框架切换 | 下拉选，切换表格/图表数据视角 |
| 运行状态 | 进度文字或 spinner |
| 主题切换 | ☀/☾ |

### 主区域
| 状态 | 显示内容 |
|---|---|
| 空状态 | 一行内联拖拽上传区（不占全屏） |
| 运行中 | spinner + 进度条 |
| 完成 | 4个指标卡 → 2个图表 → 层表格 |

### 右侧面板
| 状态 | 显示内容 |
|---|---|
| 未选层 | 隐藏 |
| 选中层 | 层名、类型、形状 → 精度指标 → 框架对比表 → 输出对比图 |

## 6. 数据类型

### ModelFile
```
id, name, format ('onnx'), size, uploadTime
```

### ComparisonTask
```
id, model, frameworks[], status, progress,
createdAt, baseline, comparisons[]
```

### LayerDiff
```
layerName, layerType (Conv/Relu/Gemm...),
inputShape[], outputShape[],
metrics[]  ← 每个框架一组指标
```

### LayerMetric
```
frameworkId,
cosineSimilarity (0~1),
maxAbsError,
meanAbsError,
relativeError,
snr,
passed (boolean)
```

---

## 7. 与后端交互（HTTP API）

```
POST /api/upload        → 上传 .onnx，返回 fileId
POST /api/tasks         → 创建任务 { modelId, frameworks[] }
GET  /api/tasks/:id     → 查任务状态 + 整体指标
GET  /api/tasks/:id/layers → 层级别精度数据
```

前端开发阶段使用 `api/mock/` 模拟数据，后端就绪后通过 `VITE_USE_MOCK=false` 切换。
